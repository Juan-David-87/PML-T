02Training Process

Mandatory standardization: Without standardization, columns like PUNTAJE_ENTIDAD (0-100) would dominate the covariance matrix over columns like QUINTIL (1-5).

Standardization (Z-score)
Each feature is transformed to mean=0 and standard deviation=1
✓ Fitted only on training data
PCA (2 components)
Eigenvectors are learned on training and transform test data
✓ No data leakage
Complete pipeline: Standardization → PCA is applied consistently to both sets
