1 · Algorithm

What is K-Means?

K-Means is an iterative centroid-based algorithm that partitions n observations into K non-overlapping clusters by minimising the within-cluster sum of squares (WCSS / inertia).

2 · Hyperparameter Configuration

Choosing the Optimal K

We swept K from 2 to 10, evaluating each candidate with two complementary criteria: the Elbow Method (inertia drop-off) and the Silhouette Score (cluster cohesion vs. separation). K = {{ results.k_final }} was selected as the best trade-off between interpretability and cluster quality.

3 · Training Process

Dataset Split & Convergence

Although K-Means is unsupervised (no labels), we apply an 80/20 hold-out split: the model is fitted only on the training set and cluster consistency is then verified on the unseen validation set via Silhouette and Davies-Bouldin scores.
